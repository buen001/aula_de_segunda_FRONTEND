Objetivos desse projeto:
Este é um projeto da disciplina Desenvolvimento Front-End para Web,
com o objetivo de usar os conhecimentos das aulas aplicadas até o presente dia
e apartir disso, desenvolver um site com base em HMTL5, CSS e JavaScrpt,
que neste primeiro momento, nesta primeira sprint não obrigatório
o uso do CSS e do JavaScrip.

O nosso site tem o seguinte tema, um site de venda de Doces Caseiros,
a loja se chama "Amor aos Pedaços", e tem como objetivo demonstrar os
doces e impulsionar as vendas de uma pequena empresa.
